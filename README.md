# Exploring Employment Trends Among Nepali Students (2023–2024)

## Project Overview
This Power BI project analyzes educational trends, entrance examination participation, academic performance, and career preferences among Nepali students.

The dashboard transforms raw student data into meaningful insights using KPI cards, slicers, and interactive visualizations.

---

## Business Objectives
- Analyze entrance examination participation trends
- Identify the most preferred educational streams
- Study gender participation trends
- Analyze students attending multiple entrance examinations
- Evaluate entrance examination performance levels
- Identify the most preferred career paths among students

---

---

## Key Insights
- IOE entrance examination recorded the highest student participation.
- Science-related educational streams dominated the dataset.
- Students attending multiple entrance exams showed high academic competition.
- Engineering and IT-related careers were the most preferred job roles.
- Average performer category contained the highest number of students.
